This is a website for people who are interested in astronomy and would like to see some data about stars visualized.

Reset amount of stars

On the left is a scatterplot of 1000 stars, with ColorIndex on the x-axis and AbsMagnitude on the y-axis. You can change the variables using the dropdown menus. The amount of stars can be changed using the slider. If you click on a star you get directed to two charts with data about that star. The currently selected star is black. The colors of the stars are an approximation of the light which it emits. Color index is used to scale the data to all the visible light (a rainbow).

There are multiple colorindices, all of which are a ratio between two intensities of light. This colorindex is the ratio between blue light and visible light. Using the colorindex of a star astronomers can approximate the temperature of that star.

The absolute magnitude (AbsMagnitude) is a way for astronomers to compare different stars. It is a number to indicate the amount of light one detects at a distance of 10 parsec (3.09 * 10^14 km).

This bar chart displays data about star 1. The barsize is scaled between the minimum and maximum of that variable in the scatterplot, with 0% being the minimum and 100% the maximum value. If you hover over the bars it shows you the absolute value of that variable.

This radar chart displays data about star 1. The value is scaled between the minimum and maximum of that variable in the scatterplot, with 0% being the minimum and 100% the maximum value. If you hover over the grey dots it shows you the absolute value of that variable.
